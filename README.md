An ESP32-based gadget with a 0.96" SSD1306 OLED 
display showing 60 animated face expressions with 
random auto-cycling. Connects to WiFi to fetch live 
time via NTP and real weather data via open-meteo.com 
with automatic location detection by IP. Includes 
RC522 RFID reader for personalised greetings, TTP223 
touch sensor to change expressions, push button for 
clock and weather screens, and an active buzzer for 
sound feedback. Fully built on a breadboard with 
custom firmware written in Arduino C++.
